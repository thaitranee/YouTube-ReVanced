<h2>Tính năng</h2>
<ul>
  <li>Xem video không quảng cáo</li>
  <li>Phát nền và khi tắt màn hình</li>
  <li>Ép độ phân giải cao hơn màn hình</li>
  <li>Giao diện tối</li>
  <li>Ẩn thẻ gợi ý và màn hình kết thúc</li>
  <li>Vuốt chỉnh âm lượng và độ sáng</li>
</ul>

<h2>Yêu cầu</h2>
<ul>
  <li>Android 7 trở lên</li>
  <li>Tài khoản Google</li>
  <li>MicroG hỗ trợ đăng nhập</li>
  <li>Cho phép cài ứng dụng từ nguồn ngoài</li>
</ul>

<h2>Cài đặt nhanh</h2>
<ol>
  <li>Gỡ hoặc tắt cập nhật YouTube gốc</li>
  <li>Cài MicroG</li>
  <li>Cài YouTube ReVanced</li>
  <li>Mở ứng dụng và đăng nhập</li>
</ol>

<h2>Lỗi thường gặp</h2>
<ul>
  <li>
    Không đăng nhập được
    <ul>
      <li>Kiểm tra MicroG</li>
      <li>Bật Google Device Registration</li>
    </ul>
  </li>
  <li>
    Ứng dụng bị văng
    <ul>
      <li>Kiểm tra kiến trúc CPU</li>
      <li>Chọn đúng phiên bản Android</li>
    </ul>
  </li>
  <li>
    Không phát nền
    <ul>
      <li>Kiểm tra quyền pin</li>
      <li>Tắt tối ưu hóa pin cho MicroG</li>
    </ul>
  </li>
</ul>
